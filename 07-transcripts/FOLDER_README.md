# 07 — Transcripts

This folder contains the journal of past Claude AI sessions where we built PrintLoop. **You don't need this to run anything.** It's here for two reasons:

1. **Context preservation** — if you ever want to ask Claude to continue working on PrintLoop, you can attach `journal.txt` to give Claude the full backstory in one read.
2. **Archeology** — when you're trying to remember "why did we choose Paystack over Flutterwave?" or "what was the original group printing spec?", the transcripts have the answers.

## What's in the journal

`journal.txt` is a chronological summary of past sessions: what we built, what decisions we made, and where each session ended. It links to longer transcript files (which are too big to attach in normal use, but exist on the system you're working with).

## When you might use this

- Onboarding a developer or a co-founder. Send them `journal.txt` so they understand the history.
- Asking Claude (or another AI) to pick up where we left off. Paste the journal at the start of the new conversation.
- Settling debates with yourself months from now ("did we already decide that?").

You can safely delete this folder if you don't need the historical context.
