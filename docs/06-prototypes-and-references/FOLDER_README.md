# 06 — Prototypes and References

Earlier prototypes from when we were still designing the system. **You don't need these to run PrintLoop** — they're here as historical reference.

## Files

| File | What it is |
|---|---|
| **kiosk-preview.html** | Early HTML mockup of the kiosk screen. Open in your browser to see what we sketched before the editorial redesign. |
| **printloop-demo.html** | An older standalone demo page. Open in your browser. |
| **PrintLoopKiosk.jsx** | An older React component for the kiosk interface. Superseded by `02-frontend/src/pages/kiosk/KioskPages.jsx`. |

## Why we kept these

- They show the design evolution, which can be useful when explaining the project to investors or new team members.
- The HTML files can be opened in any browser without any setup — useful for showing someone the kiosk concept on the spot.
- Some of the React code patterns might be useful as references when building related features later.

## Should you delete them?

You can. They're not used by anything else. If your project folder feels cluttered, delete this folder.
