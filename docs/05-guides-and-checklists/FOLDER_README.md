# 05 — Guides and Checklists

All the reference documents from earlier development phases. Most of these are **deeper dives** into specific topics — you only need to read them when you hit something the main setup manual doesn't cover.

## Read order (if you want to)

You don't have to read these. The main `PrintLoop-Setup-Manual.docx` covers everything you need for setup. These are for when you want to go deeper.

| Document | When to read |
|---|---|
| **PrintLoop_Complete_Integration_Guide.docx** | When wiring frontend ↔ backend (Part 6 of the manual). Lists every endpoint and what it does. |
| **PrintLoop_Backend_Roadmap.docx** | When planning the next phase of features. Shows what's done and what's still possible. |
| **PrintLoop_Implementation_Status.docx** | A snapshot of where the codebase was at the time of building. Useful for context. |
| **PrintLoop_Implementation_Guide.docx** | The earlier comprehensive guide. Some content overlaps with `PrintLoop_Complete_Integration_Guide.docx` — read whichever feels most useful. |
| **PrintLoop_User_Stories_Implementation.docx** | When defining new features. Matches user stories to the code that implements them. |
| **PrintLoop_Backend_Tasks_Priority.pdf** | A prioritized task list. Useful as a checklist. |
| **Kiosk_Auth_Integration_Guide.docx** | When setting up your first physical kiosk. |
| **PrintLoop_Implementation_Checklist.xlsx** | An Excel checklist of all features. Good for tracking progress. |
| **PrintLoop_User_Stories_Checklist.xlsx** | User stories in spreadsheet form. |
| **PrintLoop_Notion_Checklist.md** | The same checklist in Markdown — drop into Notion if you use it. |

## Recommendation

Open `PrintLoop_Complete_Integration_Guide.docx` first. It's the most comprehensive single document and covers backend + frontend + kiosk.
